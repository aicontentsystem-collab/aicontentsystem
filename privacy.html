5. TikTok Data Handling - Special Requirements

In strict compliance with TikTok Developer Platform Policies:

  • We access only the minimum TikTok data necessary to provide the Service (data minimization principle)
  • TikTok user data is never sold, rented, or shared with third parties for advertising purposes
  • TikTok access tokens are encrypted at rest using AES-256 encryption
  • TikTok access tokens are transmitted only over HTTPS and TLS encrypted connections
  • We do not use TikTok data to build user profiles for advertising or marketing purposes
  • We do not combine TikTok data with data from other sources to identify individuals
  • Users can revoke TikTok access at any time via TikTok Settings, Security, Authorized Apps
  • Upon revocation, all associated TikTok tokens are permanently deleted within 24 hours
